# WasmCloud Integration with KubeStellar Console

## Overview
KubeStellar Console ships a WasmCloud card providing real-time visibility into actor and provider health across your WebAssembly workloads. WasmCloud is a CNCF Incubating project building the cloud-native distributed platform for WebAssembly on Kubernetes.

## Console Integration Features
- **Actor Health Monitoring** — View actor lifecycle, status, and deployment state across clusters
- **Provider Connectivity** — Track provider availability and link status for reliable messaging
- **Multi-Cluster Visibility** — Monitor WebAssembly runtime health across multiple Kubernetes clusters simultaneously
- **Real-time Updates** — SWR (stale-while-revalidate) pattern for instant data on revisit with background refresh
