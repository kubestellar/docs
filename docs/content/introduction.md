# Welcome to KubeStellar Documentation

KubeStellar is a Kubernetes-native multi-cluster delivery system that enables you to create once and deploy many across multiple Kubernetes clusters, clouds, and edge environments.

## What is KubeStellar?

KubeStellar provides a unified control plane for managing workloads across multiple Kubernetes clusters. It offers a declarative approach to multi-cluster deployment with powerful scheduling, placement, and customization capabilities.

## Key Features

- **Multi-Cluster Deployment**: Deploy workloads to multiple clusters from a single control plane
- **Flexible Placement**: Advanced scheduling and placement policies
- **Edge Support**: Optimized for edge and resource-constrained environments
- **Kubernetes-Native**: Built on standard Kubernetes APIs and concepts
- **Customization**: Per-cluster customization of deployed workloads
